== Description ==


CSV Export will create a csv file of published and draft post data. 


== Installation ==

 Install CSV Export by the Wordpress plugin installer.
 Activate the plugin at the plugin administration page.
 Look under Tools section (Left sidebar menu) for the Export page.


== Frequently Asked Questions ==

What is exported?

 the contents of the post table (title, content, date, author, excerpt, etc.) and the custom fields and taxonomy terms
